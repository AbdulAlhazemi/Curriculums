2.  **ما ناتج الكود التالي؟**
    ```javascript
    class Animal {
      speak() {
        return "Sound";
      }
    }
    
    class Cat extends Animal {}
    
    const c = new Cat();
    console.log(c.speak());
    ```
    * A) خطأ
    * B) Sound ✅
    * C) undefined
    * D) Cat
