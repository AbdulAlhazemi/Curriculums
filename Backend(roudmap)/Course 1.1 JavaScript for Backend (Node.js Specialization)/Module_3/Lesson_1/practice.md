### 🛠️ تمرين عملي (Practice)

🧩 **المطلوب:**
1.  أنشئ ملفًا جديدًا باسم `hello.js`.
2.  اكتب فيه الكود التالي:
    ```javascript
    console.log("مرحبًا من Node.js");
    ```
3.  شغله من الطرفية باستخدام:
    ```bash
    node hello.js
    ```

> 💡 **تلميح:** تأكد أنك في نفس المجلد الذي يحتوي على الملف.
